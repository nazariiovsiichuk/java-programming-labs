# Java Programming Laboratory Works

**University coursework repository**  
Clean implementations • Consistent structure • Documented solutions

---

## ▌About the Repository

This repository contains laboratory assignments completed during a Java programming course.  
Each lab is implemented as a separate, self-contained mini-project with its own documentation and notes.

The focus is on:

- readable and structured code  
- clear implementation logic  
- consistent project organization  

---

## ▌Repository Layout

Each laboratory directory follows the same structure:

```
labX/
 ├── src/        → source code
 ├── README.md   → task description & usage
 └── report.md   → implementation notes
```

This keeps every assignment isolated and easy to review.
